# 成语猜猜猜（Python版）功能说明（按作业文档分块）

本说明严格按学校作业文档结构，分为：**基础功能、扩展功能、加分项**。

项目主文件：`idiom_game.py`

---

## 一、基础功能

### 1) 基础猜成语流程
- 随机抽取成语题目，显示该成语释义作为提示。
- 每局默认 3 次机会，逐次输入完整成语进行猜测。
- 猜对立即判定本局成功；机会耗尽后显示正确答案。

### 2) 基础交互与结果展示
- 每局展示当前题目状态、剩余机会、最终结果。
- 失败时显示成语解释（以及存在时的例句），便于理解错因。

---

## 二、扩展功能（作业 3.2 对应）

### 扩展功能一：难度选择
- 开局可选择难度：
  - 简单：隐藏 1 个字
  - 中等：隐藏 2 个字
  - 困难：隐藏 3 个字
- 通过 `mask_word` 按难度动态生成题面。

### 扩展功能二：计分系统
- 猜对基础 +10 分。
- 使用提示（显示首字）该局扣 2 分。
- 连续猜对 3 局奖励 +5 分。
- 历史最高分持久化到 `score.txt`，下次启动可读取。

### 扩展功能三：成语分类
- 支持按分类选择题库：全部/寓言/动物/人物。
- 每局可重新选择分类后进入新题。

### 扩展功能四：限时模式
- 可开启每局 30 秒倒计时。
- 超时自动判定该局失败。

### 扩展功能五：学习模式
- 猜错时显示详细解释。
- 若题库中有例句，自动一并展示。

### 扩展功能六：排行榜系统
- 新增 `leaderboard.txt` 记录玩家姓名与分数。
- 每局结束后自动刷新排行榜并展示 Top 5。
- 排行榜按分数降序排序，仅保留前五条。

### 扩展功能七：从文件加载题库
- 从 `idioms.txt` 读取成语数据。
- 支持以下格式：
  - `成语:描述:分类`
  - `成语:描述:分类:例句`
- 自动跳过空行、注释行和格式错误行。
- 若文件不存在或无有效数据，自动回退内置题库。

### 扩展功能八：双人对战模式
- 主菜单支持“单人模式 / 双人对战模式 / 创新模式”。
- 双人模式规则：
  - 两名玩家轮流按“位置 + 一个字”猜同一成语。
  - 猜对则揭示对应位置字符，先完整拼出成语者获胜。
  - 回合上限后仍未拼出则平局结束。
- 胜者分数可写入排行榜。

## 三、加分项（作业 3.3 对应）

### 1) 代码规范（代码风格）
- 将核心逻辑按职责拆分为输入、题库、计分、排行榜、对战、创新模式等函数块。
- 命名统一、结构清晰，减少重复逻辑。
- 增加单元测试文件：`tests/test_idiom_game.py`。

### 2) 异常处理（错误处理）
- 对题库、最高分、排行榜文件读写增加显式 `OSError` 提示。
- 文件格式坏行进行跳过并提示，不影响主流程继续运行。
- 非法输入不崩溃，统一引导重新输入。

### 3) 输入安全（字符串/输入安全）
- 新增范围数字输入校验：
  - `parse_int_in_range`
  - `safe_int_input`
- 新增单字符输入校验：
  - `normalize_single_char_input`
  - `safe_single_char_input`
- 对战模式中位置与字符输入均经过安全校验，避免异常路径。

### 4) 宽字符支持（Python 等效实现）
- 统一 UTF-8 文件读写，确保中文数据稳定存储与读取。
- 对战“猜一个字”按 Python Unicode 字符长度校验，支持中文字符正确比较与显示。
- 说明：作业文档给的是 C 的 `wchar_t` 方向；本项目为 Python，实现了等效的 Unicode 宽字符处理能力。

### 5) 创新功能（详细说明）
新增模式：**创新模式：释义挑战（Description Challenge）**

#### 玩法规则
- 每轮随机抽取 1 个成语，仅展示释义。
- 系统给出 4 个候选成语（1 个正确 + 3 个干扰项）。
- 玩家输入编号作答：
  - 选对：本轮 +5 分
  - 选错：显示正确答案
- 默认进行 3 轮，最终结算总分。

#### 设计与实现细节
- 主逻辑函数：`play_description_challenge_mode`
- 判题函数：`run_description_challenge_round`
- 候选项生成策略：
  - 先放入正确答案；
  - 再随机补齐 3 个不重复干扰项；
  - 最后打乱顺序输出。
- 模式结束后成绩写入排行榜并展示 Top 5，实现与主系统联动。

#### 创新价值
- 与“输入完整成语”的传统流程形成差异化玩法。
- 训练“由释义反推成语”的语义理解能力。
- 增强重复游玩兴趣，同时复用现有题库、排行榜与输入安全能力。
